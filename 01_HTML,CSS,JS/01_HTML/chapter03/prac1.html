산길

오늘 아침 일찍 산에 올랐다. 아침온도는 15º였다. 가지고 간 물의 ½를 마셨다. 한참 올라 가는 도중에 약수터를 가리키는 ← 시인이 보였다. 불현듯 양두종 님의 산길이란 시에 곡을 붙인 박태준의 ♪ 가곡이 생각났다.

                    밤에 홀로 산길을 간다
